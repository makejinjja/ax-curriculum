# Vibe Tetris (1vs1 AI Battle Update)

Claude 및 Antigravity 협업 기반(바이브코딩)으로 제작된 게임 "Vibe Tetris" 입니다. 1vs1 AI 대전 모드가 추가되어, 로컬 PC에서 혼자서도 봇과 치열한 승부를 펼칠 수 있습니다.

## 주요 추가 기능 (1vs1 대항전 모드)
- **AI 봇 대전 (Side-by-Side)**: 우측 화면에 AI 봇의 플레이가 실시간으로 렌더링됩니다.
- **고도화된 AI 휴리스틱**: AI는 빈 공간, 현재 높이, 요철 차이 등을 자체적으로 계산해 최적의 위치로 기계적인 위치를 찾아 이동합니다.
- **가비지 라인 공격 교환**: 한 쪽에서 여러 줄을 동시에 지울 시, 상대의 바닥에 회색 쓰레기 블록 줄이 융기하여 타격을 입힙니다!
  - 2줄 삭제 -> 상대방 바닥 1줄 상승
  - 3줄 삭제 -> 상대방 바닥 2줄 상승
  - 4줄 삭제(TETRIS) -> 상대방 바닥 4줄 상승!
- **레벨 업 = 타임어택**: 일정 줄을 지워 레벨업 할수록, AI의 키보드 동작 딜레이가 감소하여 후반부엔 무서운 속도의 APM(Action Per Minute)을 보여줍니다.

## 환경 설정 및 실행 방법

Windows 시스템 및 Python 3.10+ 구동 환경이 필요합니다.

1. **설치 경로 방문**
   ```cmd
   cd c:\dev\ax-workspace\tetris_project
   ```

2. **의존성 모듈 설치**
   ```cmd
   py -m pip install -r requirements.txt
   ```

3. **게임 시작!** (가로 1000px 이상의 넓은 창이 생성됩니다)
   ```cmd
   py main.py
   ```

## 게임 조작 가이드 (Player 1)

| 조작키 / 액션 | 설명 |
|---|---|
| `⬅️` `➡️` | 블록을 **좌/우**로 이동합니다. |
| `⬆️` | 블록을 **시계방향으로 회전**합니다. (충돌 시 취소 보정) |
| `⬇️` | **소프트 드롭**: 누르는 동안 블록이 조금 더 빨리 떨어집니다. 보너스 점수가 있습니다. |
| `SPACE` | **하드 드롭**: 블록 수직 최하단으로 즉각 드롭/고정시킵니다. |
| `C` 또는 `LSHIFT` | **홀드(Hold)**: 이 블록을 저장합니다 (1턴 당 한 번만 가능합니다). |
| `P` 또는 `ESC` | 게임 진행을 일시 정지(Pause) 합니다. |
